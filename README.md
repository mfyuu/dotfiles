# Zsh Startup Benchmark

This branch hosts benchmark results for [mfyuu/dotfiles](https://github.com/mfyuu/dotfiles).

## How it works

- **`data.js`** — Historical benchmark data, updated automatically on each push to `main`
- **`index.html`** — Chart UI served at [bench.mfyuu.dev](https://bench.mfyuu.dev)

Both files are managed by [github-action-benchmark](https://github.com/benchmark-action/github-action-benchmark). Do not edit them manually.
